Git to otwarto-źródłowy program do kontroli wersji. Używany głównie w konstrukcji oprogramowania.

Na początek powinniśmy ściągnąć oficjalną wersję programu `git`.
Na komputerze z `brew` będzie to:

```
brew install git
```

Na komputerze z `dnf` będzie to:

```
dnf install git
```

Aby zobaczyć podręcznik użytkownika wpisz:

```
man git
```

Aby przewijać linia po lini wciskaj `j` i `k`.
Aby przewijać stronami wciskaj `f` i `b`.
Aby wyjść wciśnij `q`.

Podstawowe komendy w git to: status, add, commit, checkout, branch, merge, rebase, push, pull.

status - służy do zorientowania się jakie zmiany wprowadzono w stosunku do ostatniej wersji na danej gałęzi
add - służy do dodania do wyróżnionych zmian
commit - służy stworzenia wersji
checkout - służy do ściągnięcia opisanej hashem lub nazwą wersji
branch - służy do zarządzania gałęziami
merge - służy do scalania zmian
rebase - służy do przepisywania historii wersji
push - służy do wypychania swoich lokalnych wersji
pull - służy do ściągania zmian innych autorów
